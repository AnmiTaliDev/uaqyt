# Uaqyt

Uaqyt (kaz. /u…Ďňąq…®t/ - time) is a minimalist Pomodoro timer Android application built with focus and productivity in mind.

## Tech Stack
- Kotlin & Jetpack Compose
- MVVM Architecture
- Hilt for Dependency Injection
- DataStore for preferences
- Compose Navigation

## Build

To build the project, run:
```bash
./gradlew assembleDebug
```
